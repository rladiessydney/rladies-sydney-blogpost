Sometimes it is nice to show the results of an R expression like weekdays(as.Date("1813-01-28")) inline. That way we find out that Pride and Prejudice was published on a Thursday. Code blocks look like this:

PPdate <- as.Date("1813-01-28")
PPday <- weekdays(PPdate)
print(PPday)
## [1] "Thursday"

An R chunk that does some basic image manipulation and plotting using the imager package:

# load packages
library(imager, quietly = TRUE, warn.conflicts = FALSE)
library(magrittr)

# specify the file path
rladieslogo <- "../1813-01-28-my-post_staticfiles/logo.png"

# specify a mapping function
map <- function(x,y) {
  list(
    x=exp(y/1000)*x,
    y=y*exp(-sin(x/60))
  )
} 

# load, warp & plot the logo
rladieslogo %>% load.image %>% imwarp(map=map) %>% plot(axes=FALSE)
